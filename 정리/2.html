2. 웹 폼
<form name="폼 이름" action="실행 응용 프로그램" method="전송 방식"> 폼으로 묶는 태그
  • name : 이름 지정
  • action : submit 누를 시 지정된 용용 프로그램 실행
  • enctype : 인코딩 방식 설정
  • method : 폼 코드 웹 서버에 전송하는 방식 "get/post"
  • target : 전송 받은 데이터 출력할 윈도우 지정

시간 입력
  • 년/월 : <input type="month" value="Default 시간 (ex : 2022-10)"

  • 년/월/일 : <input type="date" value="Default 시간 (ex : 2022-10-04)"
  • 해당 나라의 년/월/일/오전,오후/시/분/초/100분의 1초 :
    <input type="datetime-local" value="Default 시간 (ex : 2022-10-04T21:38:45.34)"
  • 시/분 : <input type="time" value="Default 시간 (ex : 21:40)"
